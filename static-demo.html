Arduino Web Communication

Static Demo - Server Required for Full Functionality

Demo Mode

This is a static demonstration of the Arduino Web Communication System. To use the full functionality with real Arduino communication, you need to run the Node.js server locally.

Server Required

The Arduino communication features require a local Node.js server to handle serial communication and WebSocket connections. This static version is for demonstration purposes only.

To Run the Full System:

  1. Clone or download this repository to your local machine
  2. Install Node.js (version 14 or higher)
  3. Open terminal/command prompt in the project directory
  4. Run: npm install
  5. Run: npm start
  6. Open: http://localhost:3000
  7. Connect your Arduino and upload the appropriate code
